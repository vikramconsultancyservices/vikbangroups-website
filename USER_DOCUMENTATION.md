### 2. Contact Form

**Location:** Contact page

**How to Fill:**

- Form clears
- "Thank you!" message appears

**On Error:**
- Error message displays below field
- Fix the issue

**Example Errors:**
- "Please enter your name" - Name field empty
- "Please enter valid email" - Email format wrong
- "Please enter valid phone" - Phone format wrong
- "Message too short" - Message less than 10 characters

## Accessibility Features

### For Screen Reader Users

**Features:**
- Proper heading hierarchy (H1, H2, H3)
- ARIA labels on buttons
- Form labels associated with inputs
- Alt text on images
- Semantic HTML structure

**Navigation:**
- Use Tab to navigate
- Use Enter to activate buttons
- Use Space to check checkboxes
- Screen reader announces current page

**Forms:**
- Tab through form fields
- Screen reader reads field labels
- Error messages announced
- Required fields marked

**Navigation:**
- Press Tab to move forward
- Press Shift+Tab to move backward
- Press Enter to activate buttons/links
- Focus visible with outline

**Forms:**
- Tab through all fields
- Enter/Space to check/uncheck
- Arrow keys for dropdowns
- Enter to submit

### For Color-Blind Users

- WCAG AA contrast ratio maintained

**Features:**

## Troubleshooting

### Issue: Page Not Loading

**Solution:**
1. Check internet connection
2. Refresh page (F5 or Cmd+R)
3. Clear browser cache
4. Try different browser
5. Disable VPN/Proxy
6. Disable browser extensions

**Contact Support:** If issue persists

### Issue: Form Won't Submit

**Solutions:**
1. Check all required fields (marked with *)
2. Fix any error messages shown
3. Verify email format is correct
4. Verify phone format is correct
5. Ensure privacy checkbox is checked
6. Try different browser

**If Still Failing:**
- Use Chat widget to report issue
- Email: info@vikban.com
- Include: Browser type, error message

### Issue: Chat Widget Not Responding

**Solutions:**
1. Refresh page
2. Check internet connection
3. Try different browser
4. Clear browser cache
5. Disable browser extensions

**Alternative:** Use contact form or email

### Issue: Colors Not Changing with Theme Toggle

**Solutions:**
1. Clear browser cache
2. Hard refresh (Ctrl+Shift+R on Windows, Cmd+Shift+R on Mac)
3. Check browser zoom (100%)
4. Try incognito/private mode
5. Update browser to latest version

### Issue: Mobile Menu Won't Open

**Solutions:**
1. Ensure you clicked hamburger menu (‚ò∞)
2. Check screen width is < 768px
3. Refresh page
4. Clear browser cache
5. Try different browser
