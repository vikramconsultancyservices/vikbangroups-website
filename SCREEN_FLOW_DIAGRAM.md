## Mobile vs Desktop Flow Differences

### Desktop (> 1024px)
- Full horizontal navigation menu
- 3-column grid layouts
- Client scroller with visible buttons
- All content visible simultaneously
- Hover effects on cards

### Tablet (768px - 1024px)
- Full navigation menu (may wrap)
- 2-column grid layouts
- Mobile-friendly scroller
- Condensed spacing

### Mobile (< 768px)
- Hamburger menu (collapsed)
- 1-column stack layouts
- Touch-optimized buttons (44px min)
- Vertical scrolling primary
- Simplified navigation
- No hover effects (tap instead)
