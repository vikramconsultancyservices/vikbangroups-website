# vikbangroups-website

## Overview
The Vikban Groups website is designed to showcase the company's AI-powered software development services. This project includes various pages that provide information about the company, its services, and contact details.

## Setup Instructions
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/vikbangroups-website.git
   ```
2. Navigate to the project directory:
   ```
   cd vikbangroups-website
   ```
3. Open the `index.html` file in your web browser to view the homepage.
